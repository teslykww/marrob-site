# MARROB — лендинг для Tilda

Готовые HTML/CSS/JS блоки под **тот же контент и порядок**, что и лендинг в `kimi2/app` (React).

## Сборка экспорта

```bash
cd marrob-tilda
npm run build
```

В папке **`tilda-export/`** появятся файлы для вставки в Zero Block, **`все-блоки-одним-файлом.html`** (только фрагменты `<body>` — в начале файла комментарий с напоминанием) и **`полная-страница.html`** / **`index.html`** (полный документ с `<head>` для просмотра в браузере). Подробная таблица порядка — в **`tilda-export/README.md`**.

**Сейчас 23 блока** (включая Hero как на kimi2, Решение со схемой PNG и зелёным CTA, преимущества с коллажем, контакты `#contact` перед FAQ, и т.д.).

## Исходники блоков (корень `marrob-tilda/`)

Редактируйте `01-header.html`, `02-hero.html`, … затем снова **`npm run build`**. В `assets/` должны быть `hero-bg.png`, `solution-scheme.png`, `logo.png`, папка `certs/` — их копирует скрипт в `tilda-export/assets/`.

## Страница дилеров

- **dealers.html** — готовая отдельная HTML-страница для B2B (партнёрская программа, условия, форма заявки). В Tilda: создайте новую страницу и вставьте содержимое dealers.html в один HTML-блок или используйте как референс и соберите страницу из блоков Tilda. Ссылка с главной: «Дилерам» в шапке и в подвале ведёт на эту страницу (укажите в Tilda актуальный URL, например `/dealers` или полный адрес).

## Настройки в Tilda

- **Формы:** в блоке 11 (форма) и на странице дилеров замените `action` формы на отправку в Tilda Forms или на ваш endpoint (например, форма Tilda с полями Имя, Телефон, Город, Площадь, Тип заявки).
- **Изображения:** в блоках Hero, Gallery, Trust и т.д. используются placeholder-URL (Unsplash). Замените на ваши изображения через медиабиблиотеку Tilda или загрузку на хостинг.

## Просмотр варианта Kimi (референс)

Чтобы сравнить с оригинальным дизайном Kimi:

1. Запустите вариант Kimi: в папке `kimi/app` выполните  
   `npx serve dist -p 5174`  
   (или из корня проекта: `cd kimi/app && npx serve dist -p 5174`).
2. Откройте в браузере **kimi-preview.html** (через ваш локальный сервер, например `npx serve . -p 3333` в папке marrob-tilda) или перейдите по ссылке «Референс (Kimi)» в подвале главной страницы.

Страница **kimi-preview.html** показывает вариант Kimi во фрейме (http://localhost:5174). Если сервер Kimi не запущен, на странице будет инструкция и ссылка для открытия в новой вкладке.
