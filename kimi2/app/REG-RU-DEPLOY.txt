Выгрузка на reg.ru (marrob.ru) — фронт + все картинки и статика
================================================================

1) Соберите проект (в папке kimi2/app):

   npm ci
   npm run build
   npm run check:dist

2) На сервер нужно залить НЕ исходники, а ЦЕЛИКОМ содержимое папки:

   kimi2/app/dist/

   Там уже есть:
   - index.html и бандлы Vite (папка assets/)
   - всё, что лежало в public/: фото, логотипы, certs/, images/, favicon, .htaccess и т.д.

3) В Cursor/VS Code (расширение SFTP): скопируйте .vscode/sftp.json.example → .vscode/sftp.json,
   вставьте пароль FTP (в git не коммитить). Для marrob.ru на reg.ru: remotePath = /www/marrob.ru
   (корневая директория сайта из панели «Сайты»), context = kimi2/app/dist.
   Протокол в примере: FTP порт 21. Если не подключается — в панели включите FTPS и в конфиге
   добавьте "secure": true (см. wiki расширения).
   Команда: выгрузить папку dist (Upload Folder / Sync Local -> Remote).

4) Пароли и sftp.json в git не коммитьте.

5) Формы: на чистом статике reg.ru нет Node/API. Заявки — через полный URL API
   (см. .env.example, VITE_LEAD_API_URL) или свой PHP-прокси на том же домене.
