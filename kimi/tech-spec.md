### 3. Problem Section (Проблема классического утепления)
**Функционал:**
- Заголовок: "Почему классическое утепление фасада — это несколько этапов и лишние риски?"
- 4 карточки проблем с иконками:
  1. Два подрядчика → разные сроки
  2. Мокрые процессы → зависимость от погоды
  3. Ошибки стыковки → мостики холода
  4. Длительный монтаж → растянутые сроки
- Подводка к решению внизу

**Анимации:**
- Заголовок: fadeIn + slideUp при скролле
- Карточки: stagger fadeIn + slideUp(40px), stagger: 0.15s
- Иконки: scale(0.8→1) с bounce easing
- Hover на карточки: translateY(-4px), border-color change

**Компоненты shadcn:** Card

## Анимации (общие)

### Scroll-triggered animations
```typescript
// Базовая анимация появления
const fadeInUp = {
  initial: { opacity: 0, y: 40 },
  whileInView: { opacity: 1, y: 0 },
  viewport: { once: true, margin: "-100px" },
  transition: { duration: 0.6, ease: [0.4, 0, 0.2, 1] }
}

// Stagger для групп
const staggerContainer = {
  initial: {},
  whileInView: {
    transition: { staggerChildren: 0.1 }
  }
}
```

### Hover animations
```typescript
const cardHover = {
  rest: { y: 0, borderColor: "#262626" },
  hover: { 
    y: -4, 
    borderColor: "#404040",
    transition: { duration: 0.3 }
  }
}
```

### Page transitions
```typescript
// Для элементов в Hero
const heroAnimation = {
  initial: { opacity: 0, y: 30 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 0.8, delay: 0.2 }
}
```

## Формы и валидация

### Поля формы
- Имя: строка, необязательное
- Телефон: маска +7 (___) ___-__-__, обязательное
- Город: строка
- Площадь: число, м²
- Тип: radio (покупка / под ключ)

### Валидация
- Телефон: минимум 10 цифр
- Показ ошибок под полями
- Блокировка кнопки при отправке

### Отправка
- POST запрос на endpoint
- Показ loader
- Success state с подтверждением
- Сброс формы
